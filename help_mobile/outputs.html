The Outputs page lists the outputs that are currently available to spend in this wallet instance.

Fields include:

  1. TX# - The transaction that this output was received by.
  2. MWC - The amount of MWC in this output.
  3. Confirmed - The number of confirmations this output has.
  4. Coinbase - Whether or not this output is a coinbase output (this will probably be NO all the time, unless you are mining to this wallet address).
  5. Commitment - The commitment associated with this output.
  6. Height - Block height of this output. This is MWC internal information.
  7. Lock height - The height after which this output may be spent.
