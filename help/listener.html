Listener

The Listener page shows the status of the underlying wallet listeners. A green light means the listener is connected, while a red light means it is disconnected.

The key function of this page is to let you generate a new MWCMQS, Tor, or Slatepack address. For improved privacy, you can use a new receiving address each time you accept funds.

To generate a new address, click Next Address to create a new one.
Alternatively, you can jump directly to a specific address index by selecting To Index.
