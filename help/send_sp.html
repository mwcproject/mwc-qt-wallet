Send Coins with Slatepack

Slatepack-based sending is an interactive 3-step process:

  1. The sender initiates a transaction and creates a Slatepack.
  2. The recipient receives the Slatepack, generates a response, and sends it back.
  3. The sender finalizes the transaction.

While this is the most advanced sending method, it is also the most private.

On this page you can specify:
– The account to withdraw MWC from
– The amount to send
– The recipient Slatepack address

To send all available MWC without creating change outputs, click the “All” button.

Transaction Proof – Contains transaction details and can be shared with others. Keeping it enabled is useful for resolving disputes, unless maximum privacy is required.

Recipient Slatepack Address – This address encrypts the Slatepack so that only the recipient’s wallet can view the transaction details. If no address is provided, the Slatepack will be sent unencrypted and can be received by any wallet.

Use Contacts to quickly select an address from your saved contact list.

Description – An optional note that will be displayed to the recipient.
